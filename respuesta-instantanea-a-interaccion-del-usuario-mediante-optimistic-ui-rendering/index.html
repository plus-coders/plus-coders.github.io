Respuesta instantánea a interacción del usuario mediante Optimistic UI Rendering (sync UI / no await UX :)

Darío Fedes (Full Stack Developer) nos comparte un ejemplo sobre cómo manejar - de manera instantánea - las respuestas a interacciones del usuario en una aplicación mediante Optimistic UI Rendering approach. Nos plantea un caso sencillo a través de una aplicación demo donde se realizan llamadas a API y éstas - como consecuencia de la latencia de la red - requieren un tiempo para procesarse.
